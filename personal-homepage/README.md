# RoCinNaN-t 的个人小站 (温暖简约风)

这是一个基于 HTML/JS 和 Node.js 开发的个人主页，采用了清爽简约的现代设计风格。

## 核心功能
- **数字分身聊天**：集成 LLM (DeepSeek) 的数字分身，可以代替我回答访客问题。
- **作品展示**：展示我的 GitHub 核心项目。
- **多功能播放器**：支持文件夹切换、搜索和播放本地音乐。
- **清爽界面**：温暖的色调，友好直观的交互设计。

## 快速开始
1. **安装依赖**：进入 `server` 目录执行 `npm install`。
2. **配置环境**：将 `server/.env.example` 复制为 `server/.env` 并填写 API Key。
3. **启动服务**：执行 `node index.js`。
4. **访问**：打开 `http://localhost:3457`。
